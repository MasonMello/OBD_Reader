What is an OBD Code

Theyâ€™re five-digit codes and are made up of four different subsets of codes that indicate where in the vehicle the malfunction lies

The letter in the DTC tells you if the code is specific to the vehicle manufacturer or a generic code that applies to all OBD-II systems. The first of these subsets is the powertrain, or P codes, and is the largest of the four categories. This is followed by network and integration (U), body (B), and chassis (C) codes.

The last three numbers tell you specific information on the vehicle system or circuit that is running into trouble.

Example: P1234
